## 7. Evidence

The primary execution evidence is the Playwright HTML report generated by running:

```bash
npx playwright test
```

The report can be opened locally using:

```bash
npx playwright show-report
```

Failure-specific evidence is configured in Playwright and may include:

* Screenshot on failure
* Video on failure
* Trace on first retry
